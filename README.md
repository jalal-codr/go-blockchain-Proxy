# go-blockchain-Proxy
This acts as gateway for the blockchain API

# Installation
go mod tidy

# Run Project 
npx @tailwindcss/cli -i ./static/input.css -o ./static/output.css --watch

go run main.go

# Build Project 
go build ./

# features 
1. Authorization 
2. Database 
3. Encryption 
4. REST API 
5. Web Sockets
6. Cashing 
7. HTTP
